# Hackathon2026
- This program uses OpenCV and a phone camera to detect cars with a camera to get the amount of cars in a particular space. This data then gets updated to a webserver hosted locally.
- To access your phone camera, download Camo on your phone and computer. Then connect your phone via USB to your computer. 
- To run this program, first you have to open 2 separate terminals for the `app.py` and `process_lot_data.py.`
- Type `python3 app.py` in the first terminal to get the webserver running. Then type in the second terminal python3 `process_lot_data.py`
- Visit: `http://127.0.0.1:5000/` in your browser to see the website.
